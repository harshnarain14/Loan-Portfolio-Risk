Project Overview

This project evaluates a 70,000-record loan portfolio to identify default risk patterns, borrower segments, and recovery-driven return metrics. Excel BI dashboards and Python-based ETL provide insights into DTI impact, purpose-level risk, ROI outcomes, and grade-based lending decisions to improve loan approval strategy.

Technologies Used

Python (Pandas, numpy, SQLite) · Excel (Pivot Tables & BI) · CSV Dataset

Project Structure

/Python_ETL.ipynb — ETL and data preparation script

/Cleaned_Data.csv — Processed dataset

/Loan Dashboard.xlsx — Excel BI dashboard

/Project Documentation.docx — Analytical findings & recommendations

How to Run
Run ETL file in Python Jupyter Notebook, then open Excel dashboard for analysis.
